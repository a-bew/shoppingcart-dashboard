# A ShoppingCart Dashboard 

## RunTime Requirement

1. Clone repo@address
   - git clone https://gitlab.com/a-bew/shoppingcart-dashboard.git
   - cd shoppingcart-dashboard

2. Json-server 
   - npm install --save-dev json-server

3. Nodemon 
   - npm install --save-dev nodemon

4. Start app@console
   - npm run dev

5. port@3000
   - http://localhost:3000
