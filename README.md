# A ShoppingCart Dashboard 

## RunTime Requirement

1. Clone repo@address
   - git clone https://gitlab.com/a-bew/shoppingcart-dashboard.git
   - cd shoppingcart-dashboard

2. Json-server 
   - npm install -g json-server

3. Start app@console
   - json-server --watch db.json

4. port@3000
   - http://localhost:3000
